# news-analyzer

Дипломный проект по окончанию курса фронтенд разработки Яндекс.Практикума. Этот сайт выполняет функцию поиска новостей по ключевому слову, а так предоставляет аналитику популярности статей на заданную тему за последние 7 дней. В проекте используется API для запроса новостей по теме. Так же присутствует страница с информацией об авторе, с разделом, где подгружаются последние коммиты на github.
